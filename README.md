# AV_Mace Engine
Game engine using SDL2, glew, assimp, imgui, glm and devil, made by the Group 4 for the Motors subject for CITM

## Controls

- WASD - Move Camera
- Shift - Accelerate movement
- Mouse Wheel - Zoom in and out
- Right Click - Rotate Camera
